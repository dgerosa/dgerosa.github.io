# dgerosa.github.io

Welcome to [Davide Gerosa](www.davidegerosa.com)'s public code repositories!

- [**precession**](https://dgerosa.github.io/precession):  Dynamics of spinning black-hole binaries with python
- [**surrkick**](https://dgerosa.github.io/surrkick): Black-hole kicks from numerical-relativity surrogate models
- [**gwdet**](https://dgerosa.github.io/gwdet): Detectability of gravitational-wave signals from compact binary coalescences
- [**filltex**](https://dgerosa.github.io/filltex): Automatic queries to ADS and INSPIRE databases to fill LaTex bibliography
- [**skywalker**](https://dgerosa.github.io/skywalker): Things I like in python
- [**spinprecession**](https://dgerosa.github.io/spinprecession):  Results on post-newtonian black-hole binary spin precession
- [**corecollapse**](https://dgerosa.github.io/corecollapse): Stellar collapse in scalar-tensor theories of gravity
- [**spops**](https://dgerosa.github.io/spops): Spinning black-hole binary POPulation Synthesis
- [**pdetclassifier**](https://github.com/dgerosa/pdetclassifier): Gravitational-wave selection effects using neural-network classifiers
