# dgerosa.github.io

Welcome to [Davide Gerosa](www.davidegerosa.com)'s public core repositories!

- [**precession**](dgerosa.github/precession):  Dynamics of spinning black-hole binaries with python

- [**filltex**](dgerosa.github/filltex): Automatic queries to ADS and INSPIRE databases to fill LaTex bibliography

- [**surrkick**](dgerosa.github/surrkick): Black-hole kicks from numerical-relativity surrogate models

- [**gwdet**](dgerosa.github/gwdet): Detectability of gravitational-wave signals from compact binary coalescences

- [**skywalker**](dgerosa.github/skywalker): Things I like in python

- [**spops**](dgerosa.github/spops): Database of population synthesis simulations of spinning black-hole binaries
