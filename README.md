# dgerosa.github.io

Welcome to [Davide Gerosa](www.davidegerosa.com)'s public code repositories!

- [**precession**](dgerosa.github.io/precession):  Dynamics of spinning black-hole binaries with python

- [**corecollapse**](dgerosa.github.io/corecollapse): Stellar collapse in scalar-tensor theories of gravity

- [**filltex**](dgerosa.github.io/filltex): Automatic queries to ADS and INSPIRE databases to fill LaTex bibliography

- [**surrkick**](dgerosa.github.io/surrkick): Black-hole kicks from numerical-relativity surrogate models

- [**gwdet**](dgerosa.github.io/gwdet): Detectability of gravitational-wave signals from compact binary coalescences

- [**skywalker**](dgerosa.github.io/skywalker): Things I like in python

- [**spops**](dgerosa.github.io/spops): Database of population synthesis simulations of spinning black-hole binaries
